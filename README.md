## Run Locally

Clone the project

```bash
  git clone https://github.com/ATLAbanana35/nitued
```

Go to the project directory

```bash
  cd nitued
```

Install dependencies

```bash
  npm i ws
  npm i uuid
```

Start the server

```bash
  cd matchmaking
  node server.js
```
